Using Barcode Separator Sheets

  1. Print Barcode Separator Sheets from a list file, or directly from an SQL database.
  2. Perform document preparation, insert separator sheets, then scan.
  3. BSCAN reads the Separator Sheet barcode, looks it up in a list or SQL database, and applies associated indexes.
  4. BSCAN exports images and index information to the imaginf database, SQL database, flat file, DiskViewer, or the Web.
